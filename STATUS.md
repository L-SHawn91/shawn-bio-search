# SHawn Bio Search Status

## Current status

SHawn Bio Search is the biomedical retrieval backbone in the SHawn research stack.

Its primary role is to:
- search papers and datasets
- normalize metadata
- enrich identifiers
- check access and local paper-store availability
- produce retrieval bundles for downstream synthesis

## Current next steps
1. standardize runtime skill deployment from the repo source
2. keep retrieval output schemas stable for downstream consumers
3. improve local paper-store reuse signals
4. validate search bundles repeatedly in real project workflows

## Operating principle
This repository should improve through repeated real-world retrieval use.
